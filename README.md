# Simple Calculator

A responsive web-based calculator built using HTML, CSS, and JavaScript.
This project performs basic arithmetic operations and provides a clean and interactive user interface for quick calculations.

---

## Features

* Perform basic arithmetic operations:

  * Addition
  * Subtraction
  * Multiplication
  * Division
* Clear all inputs (AC)
* Delete last digit (DEL)
* Toggle positive/negative numbers
* Decimal number support
* Responsive design for desktop and mobile devices
* Interactive button animations

---

## Tech Stack

* **HTML5** – Structure of the calculator
* **CSS3** – Styling and responsive layout
* **JavaScript** – Calculator logic and functionality

---

## Project Structure

```
calculator-app
│
├── index.html     # Calculator structure
├── styles.css     # Styling and layout
├── script.js      # Calculator logic
└── README.md      # Project documentation
```

---

## How to Run the Project

1. Clone the repository:

```
git clone https://github.com/yourusername/simple-calculator.git
```

2. Navigate to the project folder:

```
cd simple-calculator
```

3. Open the `index.html` file in your browser.

---

## How It Works

1. Users click number buttons to input values.
2. Operators perform mathematical calculations.
3. The equals button evaluates the expression.
4. Utility buttons allow clearing, deleting, or changing the sign of numbers.
5. Results are displayed in the calculator display screen.

---

## Future Improvements

* Add keyboard input support
* Add scientific calculator functions
* Improve animation and UI effects
